# CSHARP ASP.NET CORE 5 MVC

## Main technologies used:

- C#
- ASP.NET CORE 5 
- MVC Framework
- Entity Framework
- SQL Server

## Setup:

Create DB migration

```bash
update-database
```

Update DB migration

```bash
add-migration init
```

Delete database

```bash
drop-database
```
